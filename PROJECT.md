# MGO RKK Identity

## Версия проекта

v0.1.0

## Назначение проекта

Электронная система идентификации сотрудников и волонтеров
Московского городского отделения Общероссийской общественной организации
«Российский Красный Крест».

## Описание

Проект предназначен для создания цифровой карточки сотрудника,
доступной при считывании NFC-метки, размещенной в удостоверении.

## Текущий этап разработки

Создание базовой структуры проекта.

## Реализовано

- Создан GitHub-репозиторий.
- Настроено рабочее окружение.
- Создана структура файлов проекта.
- Подготовлена документация проекта.

## План разработки

Следующие этапы:

1. Создание интерфейса цифровой карточки сотрудника.
2. Разработка адаптивного дизайна для мобильных устройств.
3. Подключение базы данных сотрудников.
4. Настройка отображения профилей.
5. Подготовка NFC-идентификаторов.
6. Публикация первой рабочей версии.
